# ProjectBeatsRemix Version 0.3.4-alpha
A reimagination of Just Shapes and Beats and Project Arrhythmia using exclusively Python.

(Sorry for the lack of updates. As a high school student, I had to focus on exams and stuff taking up my life. I'll push an update out by the end of May.)

Controls: WASD

Supported devices: Laptop, PC

[Play Now](https://bruhnotog.github.io/ProjectBeatsRemix/)

# What's new: 4/2/2026

Debug update part 2 is out. Now, you have full control over starting points in a level with more adjustable attributes.

Type the key combination "d-e-b-u-g" while on the title screen, and a debug text should appear.
